<h1>Микросервис на aiohttp для распознавания счетов на оплату</h1>
<p>Клиент отправляет на веб-сервер изображение счёта на оплату. Сервер отправляет json с реквизитами: БИК, ИНН, КПП, номера счетов и т.д.</p>
